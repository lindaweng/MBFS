Digital Solutions Center (DSC)

My Experience



We met with all of the groups in the DSC and learned what each team did. So we learned about AIVA Chatbot, F&I Pro, Message Center, Omni Channel, DTF Website, MyMBFS, mbfs.com, and Notification Platform. We also learned about specific roles in the DSC like the Scrum Master with the Agile process, Product Owner, and UX Designer.

We also got to make our own chatbot using IBM Watson Assistant. I made mine about troubleshooting when creating a website so that my group members can ask the chatbot first before asking me. I also added responses to common complaints like "My code doesn't make sense" or "None of this is working!" and a lot of the responses lead to finding me.

I learned about bootstrap and our group made a website using bootstrap together. Through making this website we also practiced integrating code. While integrating, we learned that bootstrap has different versions and they are not compatible with each other.
